## Test Strategy

- **Fully decoupled**: All external dependencies (LLM, GPU models, APIs, file system) are replaced with mocks.
- **MockLLM**: Configurable `response_queue` for multi-turn conversation simulation.
- **MockTool**: Lightweight `BaseTool` subclass for agent tool-call testing.
- **No GPU / No API keys required**: Tests run on any machine with Python ≥ 3.10.
- **Async support**: `pytest-asyncio` with `asyncio_mode = auto` for all coroutine tests.
